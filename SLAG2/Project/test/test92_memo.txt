ゲーム仕様

- ジョーカーを除く５２枚
- 持ち点100クレジット
- １回交換可能


ワンペア    x   2
ツーペア 　 x   4
スリーカードx  10
フルハウス  x  50
ストレート  x  50
フラッシュ  x  50
ストレートフラッシュx 100


+-----------+
|           | <- オッズパネル    [OP]
|           |
|           | <- カードパネル    [CP]
|           |
|           | <- メッセージパネル[MP]
+-----------+

１）ベット
 OP:
    ベットが押されるたびにオッズの賞金が変化
 CP:
    何もない
 MP:
　  １クレジット [ベット] [スタート]

２）ディール
 OP:
 CP: 裏返しのカード５枚
 MP: GOOD LUCK

３）オープン１
 OP: 確定役があれば色替え
 CP: オープン
 MP: なし

４）チェンジ
 OP: 
 CP: カードが選択されたら裏へ
 MP: 交換するカードを選択、または、[OK]　※ＯＫ時はXXクレジット支払い

５）オープン２
 OP: 確定役があれば色替え
 CP: オープン
 MP: XXクレジット支払い
 1)へ(繰返)

プログラム

util_xxx     -- 汎用API
rdc_xxx      -- ラウンド制御--ゲーム全体動作
oddspc_xxx   -- オッズパネル制御
cardpc_xxx   -- カードパネル制御
msgpc_xxx    -- メッセージパネル制御

// util --汎用API
Hashtable $ht = util_create_msgobj(string msg, Color col);
          $ht.setmsg(string msg);     // メッセージ変更
          $ht.blink(bool onoff);      // 点滅
          $ht.setcol(Color col);      // 色
          $ht.setbgsize(int x, int y);// ＢＧのサイズを指定
          $ht.setcol(Color col);      // ＢＧ色 (透明も可能)
          $ht.go  --- GameObject;
          $ht.bhv --- slagremote_unity_monobehaviour

Hashtable $ht = util_create_card(cardmark mark,int num);           
          $ht.go  --- GameObject
          $ht.bhv --- slagremote_unity_monobehaviour
          $ht.clickFunc           --- クリック時のファンクション
          $ht.flip(bool showhide) --- フリップ
          $ht.setSelect();        --- セレクトモード
          $ht.setStrong();        --- 強調

// round control --ラウンド制御
var rdc_$sm = StateManager();
    rdc_$sm.Goto(rdc_$_Init);

function rdc_$_Init($sm,$bFirst);      --- 初期化
function rdc_$_WaitStart($sm,$bFirst); --- スタート待ち
function rdc_$_Bet($sm,$bFirst);       --- ベット
function rdc_$_Deal_1($sm,$bFirst);    --- ディール１
function rdc_$_Open_1($sm,$bFirst);    --- オープン１
function rdc_$_Change($sm,$bFirst);    --- チェンジ
function rdc_$_Deal_2($sm,$bFirst);    --- ディール２
function rdc_$_Open_2($sm,$bFirst);    --- オープン２
function rdc_$_Result($sm,$bFirst);    --- 結果

// オッズパネル制御
Hashtable oddspc_$ht = oddspc_CreatePanel();   --- オッズ一覧表示
          oddspc_$ht.SetCredit(int num);       --- オッズをnum倍にする
          oddspc_$ht.SetStrong(hand[] hands);  --- 強調する役。 NULLでOFF

// カードパネル制御
Hashtable cardpc_$ht = cardpc_CreatePanel();         --- カードパネル作成
          cardpc_$ht.deal5()                         --- 5枚配る
          cardpc_$ht.open()                          --- オープン
          cardpc_$ht.select()                        --- セレクトモード　
          cardpc_$ht.cards  : List<sting>            --- 手札

// メッセージパネル制御
Hashtable msgpc_$ht = msgpc_CreatePanel();  --- メッセージパネル作成
          msgpc_$ht.setMsg(string msg);　　 --- メッセージ
          msgpc_$ht.setCredits(int num);    --- クレジット
